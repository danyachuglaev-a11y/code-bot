# Telegram UserBot Manager

Бот для управления рассылкой в Telegram через панель управления.

## Команды:
- /start - Открыть меню
- /addtarget @username - Добавить цель
- /addgroup текст1 | текст2 - Добавить группу сообщений
- /cleartargets - Очистить цели
- /cleargroups - Очистить группы
- /setdelay 5 10 - Установить задержку
- /login +71234567890 - Войти в аккаунт
- /code 1#2#3#4#5 - Ввести код
- /password пароль - Ввести 2FA пароль
